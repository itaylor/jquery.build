Each of these tests renders the same HTML, a series of nested divs, images, inputs and buttons. that represent a simple voting application, where you vote for the best Bomberman 94 character.
The three scenarios are:
  1. jQuery.build is used to build the elements, listeners are added inline as they are created.
  2. Elements built with jQuery's DOM Builder functions, appended and bound to their listeners as the elements are added.
  3. Elements are built with native DOM manipulation, with no helper libraries.
